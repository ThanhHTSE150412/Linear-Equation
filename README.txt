# Linear-Equation
Welcome to the Katalon Studio User Guide.
  In the download folder include: 
    	+ Foder "KatalonTest": This folder contains Katalon's project for web testing in the "SourceCode" folder.
		1. After installing Katalon Studio successfully. If you don't have it installed you can go to: https://katalon.com/download/
		2. Click "Open Project" and Choose folder "KatalonTest" you download.
		3. Then you can test the web
	Note: Must run the web project's server first to avoid errors.

	+ Foder "SourceCode": In this directory contains the source code of the Linear-Equation website that we use to test.
		Run: You can use Java Web support tools to run the Project.
